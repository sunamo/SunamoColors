# NuGet Alternatives to SunamoColors

This document lists popular NuGet packages that provide similar functionality to SunamoColors.

## Overview

Color manipulation utilities

## Alternative Packages

### System.Drawing.Color
- **NuGet**: System.Drawing.Primitives
- **Purpose**: Built-in color structure
- **Key Features**: RGB, ARGB, named colors, conversions

### SixLabors.ImageSharp
- **NuGet**: SixLabors.ImageSharp
- **Purpose**: Modern image processing
- **Key Features**: Color spaces, manipulation, cross-platform

### ColorMine
- **NuGet**: ColorMine
- **Purpose**: Color space conversions
- **Key Features**: Lab, HSL, HSV, color difference algorithms

### Colourful
- **NuGet**: Colourful
- **Purpose**: Advanced color library
- **Key Features**: Multiple color spaces, chromatic adaptation

## Comparison Notes

System.Drawing.Color for basic needs. ColorMine/Colourful for advanced color science.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **System.Drawing.Color**: Built-in color structure
- **SixLabors.ImageSharp**: Modern image processing
- **ColorMine**: Color space conversions
